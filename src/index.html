compresh ~ a refreshingly easy webgl image compression tool ~ v0.1 alpha
0% Drop .png images anywhere

Drop your .png files in this window to convert to webgl compatible compressed texture formats

  • DXT as .dds
    • Also known as S3TC
    • API Level: WebGL 1.0
    • Devices commonly used on: PC and OSX
    • Images without transparency will be exported as: myimage.dxt1.dds
    • With transparency: myimage.dxt5.dds
  • PVR1 as .pvr
    • API Level: WebGL 1.0
    • Devices commonly used on: iOS
  • ETC1 as .ktx
    • API Level: WebGL 1.0
    • Devices commonly used on: Android
    • If alpha is present then an extra image file is created with the naming convention myimage.etc1-alpha.ktx
  • ETC2 as .ktx
    • API Level: WebGL 2.0, GLES 3.0, and OpenGL 4.3
    • Devices commonly used on: Android
    • myimage.etc2.ktx
  • ATC as .dds
    • Also known as ATITC
    • Can't find a ATC converter for mac
    • Devices commonly used on: Android with Qualcomm chipsets
    • Supported by: WebGL 1.0
    • myimage.atc.dds
  • Lossy PNG
    • Fallback for canvas mode
    • Compressed using pngQuant
  • ASTC as .astc
    • API Level: No WebGL support, GLES 3.2, and OpenGL 4.3
    • Devices commonly used on: All modern devices as of 2016
    • This is the best texture format available for native games/apps. Most modern gpus and phones support this format. Currently there are no official plans from Khronos group to make it part of any future WebGL specifications.
    • File name example (6x6 is the block size): myimage.6x6.astc

Options:

All options are currently disabled, but the default automatic settings will still work.

  • No mip-maps
  • Resize to power of 2
  • Resize to be square
  • Premultiply transparency
    • In photoshop this is the equivalent making a group with the transparency as a mask and putting the main layer in the group, along with a black layer at the bottom of the group.
  • Get gzip size
